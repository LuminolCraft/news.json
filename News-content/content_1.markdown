## 常用指令公告

欢迎游玩 LuminolCraft 服务器！以下是玩家常用的指令列表，快速了解并提升你的游玩体验！

___**注意**: 带 ___`$`___ 标记的命令参数是可选的，请根据需要使用！带括号___ `()`___的标记的表明该命令只有在其指定的服务器才能使用！___

---

### 传送相关指令

- **/tpa [玩家]** - 向某位玩家发送传送请求。
- **/tpahere [玩家]** - 向某位玩家发送到自己的传送请求。
- **/tpaaccept $[玩家]** - 接受最新的传送请求。如果指定玩家，则接受该玩家的请求。
- **/tpadeny $[玩家]** - 拒绝最新的传送请求。如果指定玩家，则拒绝该玩家的请求。
- **/tpacancel $[玩家]** - 取消发送的最后一次传送请求。如果指定玩家，取消对该玩家的请求。
- **/back** - 将玩家传送到最后一次死亡地点。

### 家园相关指令

- **/sethome <自己设定的名称>** - 将当前位置与指定名称绑定。

  - 示例: `/sethome home1` - 将当前位置绑定为 `home1`，名称可任意设置。
- **/home <自己设定的名称>** - 传送到指定名称绑定的位置。

  - 示例: `/home home1` - 传送至之前绑定为 `home1` 的坐标。
- **/delhome <自己设定的名称>** - 删除指定名称的传送坐标。

  - 示例: `/delhome home1` - 删除 `home1` 的坐标点位，之后无法通过 `/home` 传送。

### 领地相关指令和使用方法（仅综合生存）

- 创建领地前先合成一个 `木锄`然后用 `木锄`左键您所需要创建领地的第一个选区，随后用 `木锄`右键你需要创建领地的第二个选区，确保您所选择的两个选区构成的长方体框住了所有您需要创建领地的地方，随后使用 `/res create <你创建领地的名字>`创建领地，创建领地每个方块会收取您 `0.2`点券。
- `/res create [领地名]` — 创建一个领地。
- `/res remove [领地名]` — 删除一个领地。
- `/res subzone [子领地名]` — 在所处的领地中创建一个子领地。
- `/res auto [领地名] [范围]` — 自动在你站着的位置为中心，创建你能创建最大的领地。
- `/res select [x] [y] [z]` — 选取以你为中心需要被保护的区域，（Z=高度，X/Y=横轴）。
- `/res select size` — 显示所选区域大小。
- `/res select cost` — 显示已选择区域价格。
- `/res select auto [玩家]`  — 打开自动选择区域。
- `/res select expand [数值]` — 选区向面对的方向选择扩展数值大小。
- `/res contract <领地名> [缩小单位]` — 从你面对的方向缩小领地。
- `/res select shift [数值]` — 选区向面对的方向平移。
- `/res select vert` — 高度从天空扩展到基岩。
- `/res select sky` — 高度扩展到天框。
- `/res select bedrock` — 高度扩展都基岩。
- `/res select chunk` — 选择目前所在区块。
- `/res area add [领地名] [区域ID]` — 为领地添加物理区域。
- `/res area remove [领地名] [区域ID]` — 移除领地的物理区域。
- `/res area remove [领地名] [区域ID]` — 替换领地的物理区域。 (可以与其他领地区域重叠)
- `/res setmain [领地名]` — 设置主领地。

### 箱子商店使用指南和相关指令（仅综合生存)

> 在大多数情况下，你**不需要输入任何指令**即可完成创建箱子商店、物品买卖等等，大多数的操作都通过简单的点击完成。

#### 如何创建商店

1. **准备容器** ：在地上放置一个箱子、陷阱箱或潜影盒 。
2. **选定物品** ：手持你想要出售的物品，左键点击箱子 。
3. **设定价格** ：在聊天框直接输入单价（支持简写，如输入 `2.5k` 代表 **2500**） 。
4. **补充库存** ：将货物放入箱子内，商店即刻开始营业。

#### 如何购买商品

1. **查看信息** ：左键点击商店招牌或箱子，聊天框会显示该商品的详细信息，包括物品名、单价、库存、附魔属性等等 。
2. **发起购买** ：右键点击商店招牌 。
3. **输入数量** ：在聊天框直接输入你想要购买的数量（无需加斜杠），系统将自动结算并扣除点券。

#### 商店管理

> 一旦商店创建成功，你可以通过**右键点击商店招牌**打开“商店控制面板”GUI 进行快捷管理。

* **模式切换** ：
  * **出售模式**：默认模式，玩家向你买东西。
  * **收购模式**：你向其他玩家买东西。使用指令 `/qs buy` 可快速切换 。
* **图形化管理** ：在控制面板中，你可以直接修改价格、增加/删除商店员工、转让商店所有权或一键撤销商店。
* **销售统计** ：使用 `/qs history` 查看近期交易记录。你可以看到“最有价值客户”（消费最高的玩家）以及过去 **24** 小时至 **30** 天内的详细销售额统计。
* **收益分成** ：你可以通过 `/qs benefit` 将好友添加为合作伙伴，系统会自动在每笔交易成交时按比例为多名玩家分账。

#### 常用指令

> 注：`[]` 表示必填参数，`<>` 表示选填参数。

* `/qs find [物品名]` - 搜索附近正在交易该物品的商店，系统会为你指引方向。
* `/qs suggestprice` - 查看全服内同类商品的市场参考价（最高、最低、平均及中位价格） 。
* `/qs list <页码>` - 分页查看你拥有的所有商店列表及其详细坐标 。
* `/qs history [owned/accessible]` - 打开交易历史 GUI，查看销售统计和最有价值客户（`owned`表示自己的商店，`accessible`表示你有权访问的商店数据） 。
* `/qs benefit add [玩家] [比例]` - 添加收益合伙人，每笔交易自动按设定的百分比分成。
* `/qs name [名称/none]` - 为商店设置一个自定义名称，输入 `none` 可清除名称 。
* `/qs sign [材质]` - 修改当前商店招牌的木材种类（例如 `oak_sign`, `spruce_sign`，分别为橡木告示牌、云杉木告示牌）。
* `/qs discount install [代码]` - 激活特定的折扣码，在购物时享受价格减免 。
* `/qs fetchmessage` - 手动从服务器拉取最近错过的商店交易通知 。
* `/qs help <子指令>` - 查看特定指令的详细帮助说明。

#### FAQ 常见问题解答

**Q：为什么我左键点击箱子没反应，无法创建商店？**

* **权限保护** ：请检查你是否在他人领地，或者该区域是否开启了“禁止贸易”标志。
* **黑名单物品** ：请检查服务器是否禁用某些物品售卖，这些物品无法创建商店 。

**Q：箱子里明明有货，为什么提示“库存不足”？**

* **NBT 匹配检查** ：箱子商店会严格检查物品的 NBT 数据。如果你箱子里的物品耐久度、附魔顺序、甚至自定义名称与创建时手持的物品有任何细微差别，系统都会拒绝交易。建议店主清空箱子后重新放入完全相同的物品。

**Q：我想做一个既能买又能卖的商店可以吗？**

* **可以**。你需要放置两个相邻的箱子，分别创建一个“出售”商店和一个“收购”商店。通过这种方式，玩家可以向其中一个箱子出售物品，并从另一个箱子购买 。

**Q：我的商店招牌被别人改了/洗了，怎么办？**

* 插件默认会锁定商店招牌以防止修改。你可以使用 `/qs sign` 随时更换招牌材质。如果发现有人能破坏你的招牌，请及时联系服主检查领地权限设置。

**Q：如何给朋友权限让他帮我补货？**

* 你可以通过商店控制面板 GUI 添加员工，或者使用指令（如果你更喜欢的话）。员工可以协助补货或取货，但无法修改价格或通过 GUI 界面管理商店。

### 经济系统相关指令（仅综合生存）

- **/sell** - 售卖物品指令。
  - 示例：`/sell hand` - 售卖您手上的物品。
- **/baltop** - 查看经济排行榜。
- **/pay [玩家名] [点券数]** - 向该玩家转账点券。
  - 示例：`/pay NaT_Jerry 100` - 向 NaT_Jerry 转账 `100` 点券。

### 其他实用指令

- **/rtp** - 将你随机传送到当前世界的某个位置。
- **/sf guide** - 获取 Slimefun 指南书，右键点击打开，查看所有分类。这是 Slimefun 游玩中所需的第一件物品（仅综合生存)。
- **/tpsbar** - 显示一个 bossbar，展示当前的 TPS/MSPT 数据。
- **/skin [皮肤名]** - 设置你在服务器内的皮肤。
- **/shout** - 向全服发起喊话（亦可以在发送内容前加 `!`代替）。
  - 示例：`/shout 您要发送的内容`或 `! 您要发送的内容`
- **/menu** - 打开菜单，返回大厅或切换路线（仅纯净生存）。
